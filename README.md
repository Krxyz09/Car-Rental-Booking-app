# Car-Rental-Booking-app

A full-stack Car Rental Booking application built with **React**, **Express.js**, **MongoDB**, and **ImageKit**. The platform enables users to browse available vehicles, view detailed car information, upload and manage car images efficiently, and make rental bookings through a modern and responsive interface.

## Features

* 🚗 Browse and search available rental cars
* 📋 View detailed car specifications and pricing
* 📅 Book cars for selected rental periods
* 👤 User authentication and authorization
* 🖼️ Optimized image storage and delivery using ImageKit
* 📱 Fully responsive user interface
* ⚡ Fast and scalable REST API built with Express.js
* 🗄️ MongoDB database for efficient data management

## Tech Stack

### Frontend

* React
* React Router
* Axios
* Tailwind CSS

### Backend

* Express.js
* Node.js
* JWT Authentication
* MongoDB & Mongoose

### Media Management

* ImageKit for image upload, optimization, and CDN delivery

## Project Overview

The Car Rental Booking App streamlines the vehicle rental process by providing a seamless experience for both customers and administrators. Users can explore available cars, check pricing and availability, and make bookings, while administrators can manage vehicle listings and rental operations. ImageKit ensures fast image loading and optimized media delivery across devices.
